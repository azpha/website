HiveMC Stats Lookup

What happened?

On April 15th, 2021 at 5pm EST // 10pm BST, the HiveMC shut down their Java Edition servers. Their API is up until sometime in 2022, which still has accessible player data.
This is complicated for most users, so I decided to make something that can look up this data.
Note: I do not have a way to save all of this data at the moment.
It is recommended you save it yourself via the provided raw JSON link BEFORE 2022.
